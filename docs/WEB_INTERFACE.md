# MBS匹配系统 - Web界面

## 🎨 界面特色

### 现代化设计
- **响应式布局**: 支持桌面和移动设备
- **Tailwind CSS**: 现代化的UI组件库
- **动画效果**: 平滑的过渡和加载动画
- **直观操作**: 拖拽式项目选择和实时反馈

### 置信度可视化
- **颜色编码**: 绿色(高) > 黄色(中) > 橙色(低) > 红色(很低)
- **进度条**: 直观显示置信度百分比
- **排序功能**: 支持按置信度升序/降序排列
- **实时更新**: 动态显示匹配结果

## 🚀 快速启动

### 方法1: 使用演示脚本
```bash
# 启动Web界面演示
python3 demo_web.py
```

### 方法2: 手动启动
```bash
# 启动服务器
python3 run_server.py

# 打开浏览器访问
http://localhost:8000
```

## 🎯 核心功能

### 1. 智能搜索
- **实时搜索**: 输入症状描述即可获得项目建议
- **上下文感知**: 根据设置、提供者类型等调整结果
- **置信度排序**: 按匹配度从高到低排列结果

### 2. 项目选择
- **多选支持**: 可同时选择多个项目进行验证
- **可视化反馈**: 选中的项目以标签形式显示
- **快速移除**: 点击标签上的×号快速移除项目

### 3. 实时验证
- **即时验证**: 选择项目后立即进行规则验证
- **详细结果**: 显示可计费项目、被拒绝项目、冲突信息
- **原因说明**: 提供详细的验证失败原因

### 4. 置信度排序
- **智能排序**: 默认按置信度降序排列
- **切换排序**: 点击排序按钮切换升序/降序
- **视觉指示**: 排序图标显示当前排序方向

## 🎨 置信度可视化

### 颜色编码系统
```
🟢 绿色 (80-100%): 高置信度匹配
🟡 黄色 (60-79%):  中等置信度匹配
🟠 橙色 (40-59%):  低置信度匹配
🔴 红色 (0-39%):   很低置信度匹配
```

### 进度条显示
- **动态宽度**: 进度条宽度对应置信度百分比
- **渐变色彩**: 从红色到绿色的渐变效果
- **平滑动画**: 宽度变化带有平滑过渡效果

## 🔧 技术实现

### 前端技术栈
- **HTML5**: 语义化标记
- **Tailwind CSS**: 实用优先的CSS框架
- **Vanilla JavaScript**: 原生JavaScript，无框架依赖
- **Font Awesome**: 图标库

### 后端集成
- **FastAPI**: 异步Web框架
- **静态文件服务**: 自动提供HTML/CSS/JS文件
- **CORS支持**: 跨域请求支持
- **实时API**: RESTful API接口

### 响应式设计
```css
/* 移动设备 */
@media (max-width: 768px) {
  .grid-cols-1 { /* 单列布局 */ }
}

/* 桌面设备 */
@media (min-width: 1024px) {
  .lg:grid-cols-2 { /* 双列布局 */ }
}
```

## 🎮 使用指南

### 基本操作流程
1. **输入症状**: 在左侧面板输入患者症状描述
2. **设置参数**: 选择适当的设置、提供者类型等
3. **搜索项目**: 点击"搜索项目"按钮
4. **查看结果**: 浏览按置信度排序的项目建议
5. **选择项目**: 勾选感兴趣的项目
6. **验证申报**: 点击"验证申报"按钮
7. **查看结果**: 分析验证结果和冲突信息

### 高级功能
- **置信度排序**: 点击排序按钮切换排序方式
- **批量选择**: 使用复选框快速选择多个项目
- **实时统计**: 查看系统统计信息
- **状态监控**: 监控系统健康状态

## 🚀 性能优化

### 前端优化

## 🔍 故障排除

### 常见问题
1. **无法访问界面**: 检查服务器是否启动
2. **搜索无结果**: 尝试更通用的关键词
3. **验证失败**: 检查项目编号是否正确
4. **样式异常**: 清除浏览器缓存
