### 2. 混合搜索算法

#### 分数融合
```python
def combine_search_results(tfidf_results, semantic_results, top_k):
    # 创建项目映射
    result_map = {}
    
    # 添加TF-IDF结果
    for result in tfidf_results:
        result_map[result.item_num] = {
            'tfidf_score': result.score,
            'semantic_score': 0.0
        }
    
    # 添加语义结果
    for result in semantic_results:
        if result.item_num in result_map:
            result_map[result.item_num]['semantic_score'] = result.score
        else:
            result_map[result.item_num] = {
                'tfidf_score': 0.0,
                'semantic_score': result.score
            }
    
    # 计算混合分数
    hybrid_score = 0.3 * tfidf_score + 0.7 * semantic_score
```
